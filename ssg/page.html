Static Site Generator

siiky

2017/03/06

EDIT: WIP.

Simple set and forget static site generator

  • Markdown -> HTML (keep overall look of markdown)
  • Filesystem structure traversing (tree like)

0.1 Markdown -> HTML

  • Headers
  • Code (blocks and inline)
    • Blocks
    • Inline
  • URLs (inline and by reference)
    • Inline
    • Reference
  • Images (inline and by reference)
    • Inline
    • Reference

0.1.1 Headers

From:

# H1
## H2
### H3
#### H4
##### H5
###### H6

to:

<H1># H1</H1>
<H2>## H2</H2>
<H3>### H3</H3>
<H4>#### H4</H4>
<H5>##### H5</H5>
<H6>###### H6</H6>

0.1.2 Code

0.1.2.1 Blocks

From:

```some\_lang\_here
some code here

to:

```html
```some\_lang\_here<CODE>
some code here
</CODE>```

0.1.2.2 Inline

From:

`some code here`

to:

`<CODE>some code here</CODE>`

0.1.3 URLs

0.1.3.1 Inline

From:

[some text](some_url)

to:

<A HREF="some_url">some text</A>

0.1.3.2 Reference

From:

[some text][ref]
[ref]: some_url

to:

<A HREF="some_url">some text</A>

0.1.4 Images

0.1.4.1 Inline

From:

![some text](some_url)

to:

<IMG SRC="some_url" ALT="some text">

0.1.4.2 Reference

From:

![some text][ref]
[ref]: some_url

to:

<IMG SRC="some_url" ALT="some text">

0.2 Filesystem Structure Traversing

From:

.
├── words
│  ├── slpod.md
│  └── ssg.md
└── index.md

to:

.
├── words
│  ├── slpod.md
│  ├── slpod.html
│  ├── ssg.md
│  └── ssg.html
├── index.md
└── index.html

Inside index.html:

.
└── words
├── <A href="words/slpod.html">slpod.md</A>
└── <A href="words/ssg.html">ssg.md</A>
